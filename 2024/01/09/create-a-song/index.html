使用 ChatGPT + Suno 写一首小歌

发表于 2024-01-09 | 更新于 2024-01-10 | | 阅读次数

昨晚上做梦，梦见学校的一个晚自习时间，几个小伙伴来唱歌，突然来了一个胖胖的学弟，开启麦霸模式，唱了一首歌，那饱满的情感，朴实真诚的歌词，掷地有声的演唱方式，瞬间变成了学校演唱会。

醒来后，我还隐隐那首歌记得最后一句好像是：「最重要的是，我从未改变过你」。

想到前两天在 Twitter 上看见遁一子使用「ChatGPT + Suno」创建过音乐。我也抓住情绪的尾巴，用 ChatGPT 辅助写了歌词，再用 suno.ai 生成了一首小歌。

歌词：

ChatGPT 生成的歌词还是太浅显直白了，基本不可用，可能是我提示词没写好。Suno 生成的第一首歌感觉最好，就是最后这句没有唱完，婆娑的娑 suo 发音错了。
