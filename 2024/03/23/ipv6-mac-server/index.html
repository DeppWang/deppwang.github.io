使用 IPv6 免费把自己电脑变成 WEB 服务器

发表于 2024-03-23 | 更新于 2024-03-23 | | 阅读次数

我们经常有这样的需求，需要部署点应用，使其可以 WEB 访问，但又不想买服务器，可能这个应用不暴露到公网访问，就自己用。有的场景我们可以使用 Vercel，如搭建个 ChatGPT 网页服务，但有的场景又不太适合用 Vercel，如你需要跟谁联调点东西。这时候，你可以利用 IPv6 把自己电脑变成服务器用，只需要找到电脑的 IPv6 公网 IP与启动的服务的端口，配置指定主机 IP 可以通过 IPv6 公网 IP + 端口访问服务就可以了。如果 IP 经常变，还可以通过 DDNS（如ddns-go ）和 Cloudflare DNS 配置，实现域名 + 端口访问服务。PS：我原来买了阿里云的轻量服务器，简直不能用，动不动就直接卡死，就需要重新启动。

一、获取自己电脑的 IPv6 公网 IP 并配置可被访问

现在宽带都支持 IPv6，你可以让你家里每个设备都获得一个 IPv6 公网 IP，如果你没有，需要你将光猫拨号改为路由器拨号，再用一个支持 IPv6 的路由器，这篇文章有讲如何设置 IPv6。

当你宽带有了 IPv6，可以通过这个命令获取自己电脑的 IPv6 公网 IP。

返回结果一般如下，红包部分就是你的 IPv6 公网 IP

获取了公网 IP，还需要设置此 IP 可被指定 IP 访问。我使用软路由，配置示例：在「通信规则」中配置「240e:xxx:db2f」可访问自己电脑的 9090 端口。如果你不使用软路由，可能尝试用路由器的 DDNS 功能。
