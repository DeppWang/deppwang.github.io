我学习使用五笔的经验

发表于 2022-06-10 | 更新于 2023-12-30 | | 阅读次数

我是 2021 年短暂学习过五笔，但一直没有一个比较宽松的时间，没有学会。最近趁换工作间隙重新捡起来，通过这一段时间的学习与使用，目前工作基本可以使用五笔了，算是出师了。遂分享一下我学习使用五笔的经验。

这篇文章没有非常详细的从 0 到 1 介绍五笔，如果你从其他地方了解过五笔，可能看起来更加无障碍。个人认为最值得阅读的是「 字根规则理解（98 版）」部分。

我学习五笔有这几个原因：

  1. 搜狗拼音输入法的隐私问题。搜狗输入法的联想功能确实很智能，系统自带输入法还不够智能，不用搜狗打字效率比较低
  2. 不想选词。拼音输入法需要经常选词，五笔重码率低，不常选词
  3. n l 不分、h f 不分、前鼻音后鼻音不分（静 jing 心、尽 jin 力），影响打字效率。五笔可以提升打字速度。（我重庆人，小时候自己语文没学好）
  4. 打五笔不容易提笔忘字

原来也想过学习双拼，但双拼的重码率，以及 n l 不分，不能完全解决问题。遂一步到位，直接五笔。

个人认为五笔属于核心技术掌握自己手上的一门技能，可以避免「卡脖子」。学习使用后可以每天受益，如果你也有我这种困扰，推荐学习使用。

学习五笔唯一困难的地方就是得有一段时间的过渡期，一段不追求打字速度的时间。不然遇到需要快速打字的时候很难受。

个人认为学习五笔需要 3 个前置条件：

  1. 英语盲打。不会英语盲打的话，字根肌肉记忆可能更费时一些
  2. 拇指上屏。习惯拇指上屏，而不是指定数字上屏，切换五笔会更连贯一点
  3. 有一段不追求打字速度的时间

学习路线

学习五笔不用背字根口诀，也不用专门专项训练单字或词组。靠背字根口诀来记忆字根位置，成本高、效率低，容易放弃。五笔到最后完全是肌肉记忆，但如果一开始就是想通过训练单字或词组达到肌肉记忆，很难，也可能中途放弃。

开始学习五笔时，需要理解字根位置规则，逐步理解记忆所有字根位置，这样可以快速上手，建立信心，在使用中逐渐提升速度，形成肌肉记忆，从而学会五笔。我将在下文分享我理解记忆所有字根位置的方式。

学习五笔分为两阶段：

  1. 规则理解
    • 字根规则理解。理解记忆所有字根位置
    • 拆字规则理解
  2. 使用
    • 在使用中提升速度，形成手指肌肉记忆
    • 最终效果，看见一个字（词组）不去想什么键，手指就打出来了

字根

五笔指将汉字笔画归纳为五种基本笔画：横、竖、撇、捺（点）、折

  • 提笔，从左到右，所以为横

五笔如果完全使用笔画，那么按键次数太多。 每个字可以拆解为多个部分，王永民老师将其研究后，提取常用部分，分为一个个字根。每个汉字由多个字根组成。

分区

横竖撇捺折（12345），分别在 「左、右、左上、右上、下」 这 5 个区域

86 版与 98 版选择

不用新世纪，字根太少，学习容易熟练难

86 版与 98 版区别

字根不同

  • 86 版字根小一点，一般称为（小）字根，98 版一般称为（大）码元
    • 示例：“皮无又” vs “皮”；98 版才有的：“甫”、“夫”、“未”等
  • 两者部分字根位置不同。98 版调整了部分字根位置
    • 如：“广”、“舟无点”；“刀”、“臼”（按笔画顺序调整）

拆字不同

  • 拆字顺序：86 版按从左到右，从上到下拆字；98 版按笔画顺序拆字
    • 桃：木 + 四点 + 儿 vs 木 + 儿 + 四点
    • 非：三 + || + 三 vs | + 三 + | + 三
  • 98 版拆字更合理一点。86 版 vs 98 版
    • 拆字根合理。余：人 + 禾 + 识别码 vs 人 + 一 + 朩；离：亠 + 乂 vs 文
    • 拆字合理。86 版「取大优先」不满足。求：十 + 水无亅 + 丶 vs 一 + 水 + 丶

98 版因为字根更大，适合词组输入。86 版五笔拆字不规范减少了单字重码，使得单字重码率随之下降，86 版适合单字。

我先学的 86 版，但因为拆字顺序，我选择了 98 版，因为在拼音更多使用词组，我现在使用 98 版词组输入。你可都尝试用一下，再决定选择哪一种。

规则理解

字根规则理解（98 版）

字根规则

  1. 分区。横竖撇捺折（12345），左、右、左上、右上、下
  2. 前两笔组合。横横 G 11 (“王”)、横竖 F 12 (“十”)、横撇 D 13 (“厂”)、横捺 S 14、横折 A 15 (“七”)
  3. 前几笔重复。横 (“一”) 1*1 G 11、横横 (“二”) 1*2 F 12 、横横横 (“三”) 1*3 D 13
  4. 形与意。示例：“五”（形 “王”）；“学头”、“小 “（3 点）；“亻”（意 “人”）
  5. 联想记忆。示例：“口”(kou) K 23；“廾” 132，3+2=5，A 15
  6. 强记。木丁西、牛巴马

拆字规则理解

拆字原则

  • 能散不连：好 = 女 + 子
  • 能连不交：天 = 一 + 大，而不是 二 + 人。
  • 取大优先：
    • 克 = 古 + 儿，而不是 十 + 口 + 儿。
    • 求 = 一 + 水 + 丶，而不是 十 + 水无亅 + 丶
    • 墙 = 土 + 土 + 丷 + 口，而不是 土 + 十 + 关无大 + 口
    • 一小二大：
      • 万 = 一 + 万无一，而不是 丆 + 折
  • 书写顺序：
    • 出 = 凵 + 山，而不是 山 + 山；
    • 里 = 日 + 土，而不是 田 + 土；
    • 单 = 丷 + 日 + 十，而不是 丷 + 田 + 十；
    • 字根笔画前两笔是连续的
      • 为 = 丶 + 力 + 丶 ，而不是 两点 + 力；
      • 非 = I + 三 + I + 三，而不是II + 三 + 三

特殊

  • 戍：awi，不是 dn
  • 云：与 “去” fcu 相同，可自定义 fcul
  • 截：fawy，十 + 戈，因为 戈 的横的笔顺在前，所以 fa

四码输入

  • 字根大于 4 个：1 2 3 + 最后字根。如：蔻 apfc
  • 字根小于 4 个：最后加一个识别码。识别码：最后一笔 + 字型代码
    • 字型代码：字体结构，左右结构 1、上下结构 2、杂合（组合）结构 3。理解记忆：正常上下左右，反着来，左右上下
    • 示例：元 = 二 + 儿，最后一笔「折」 5， 上下结构 2， 识别码 52 -> B。所以 元 = fqb
  • 本身就是字根
    • 笔画不超过 4 笔：先字根 + 笔画输入。“川”：kthh
    • 笔画超过 4 笔：先字根 + 笔画输入 + 最后一笔。“甫”：sghy；“羊”：uyth

词组输入

  • 两字词语：检验 = “木” + “人” + “马缺一” + “一”。12,12，swcg
  • 三字词语：老古董 = “土” + “古” + “艹” + “丿”。1,1,12，fdat
  • 四字成语：天天向上 = “一” + “一” + “丿” + “上” 。1,1,1,1，ggth
  • 多于四字的短语：中华人民共和国 =“口” + “亻” + “人” + “口”。1,1,1,*,1，kwwk

词组需配置词库

使用

输入法选择

  • macOS Big Sur 及以上版本自带五笔（五笔设置）
    • 词库更丰富
    • 自动调节词频
    • 词库也可自定义，可自动同步，Link
  • macOS 清歌输入法
    • 可配置五笔 + 拼音混合输入
    • 建议不使用「用户词库」功能，而使用「自定义主词库」，自定义主词库放 iCloud 文件夹，这样就能使用 iCloud 同步个人词库。ps：增删词组后，每次需重新「生成索引」；「五笔拼音混合」下需开启「逐码提示」才能「直接上屏」
  • 98 五笔输入法 没用过，mark

单字 vs 词组

  • 全单字：所有字都 4 码。不足 4 码，空格替代。单字适合打长句，加词组麻烦
  • 全词组：词库无底洞
  • 单字 + 小词库：词库是常用的两字词语。我的选择。常用的两字词组要全，否则单字、词组切换频繁会影响效率

辅助工具

  • iOS：五笔打字练习。手机练习无法增强手指肌肉记忆，不建议购买
  • Web：岩兔五笔练习（86 版）
  • 拆字 APP：iWubi APP，已开源
  • 笔画顺序：https://bihua.bmcx.com/
  • 打字速度测试：https://dazi.kukuw.com/、10fastfingers.com

心得

对我来说，耗时较多的部分：1、前期字根联想记忆（5 天左右）；2、每个字怎么写；3、每个字怎么拆。从字根学习到基本工作可用，差不多 1 个月时间。

我使用清歌输入法，开始我一直只使用五笔，关闭「逐码提示」，使自己能更好练习。但在偶尔需要快速打字时，反尔成为桎梏。再设置使用「五笔拼音混合」、「逐码提示」后，更加流畅输入，体验感更好。但后期需要设置为五笔输入，避免过度依赖拼音输入

在开始学习使用五笔打字时，每一次敲打键盘，都带有一点成就感，可能是学习新技能才特有的感受吧。这是学习五笔最有趣的地方。
