推荐几个可以提高程序员生存技能的效率软件

发表于 2018-09-17 | 更新于 2020-05-08 | 分类于 Software & Website | | 阅读次数

把我常用的效率软件推荐一波。分别是：IDEA（集成编译环境）、ShadowsocksR（科学上网）、Wunderlist（奇妙清单） 和 Wox（Windows 效率启动器）.

IDEA（集成编译环境）

我认为 IDEA 是开发 Java 项目最好的集成编译环境（IDE），属于用了就回不去的软件。

点此 下载地址，下载最终版 (Ultimate)。

安装完成后，启动 IDEA，在如下界面选择 License server 输入：http://idea.iteblog.com/key.php ，如果链接失效，可参考这个网址： http://idea.lanyus.com

关于 IDEA 的常用快捷键、插件及设置，可参考这篇文章： https://depp.wang/2018/08/14/IDEA's-common-shortcuts-plugins-and-settings/

ShadowsocksR（科学上网）

对于程序员来说，需要使用 Google 等国外网站，所以科学上网也成了必备技能。推荐使用 ShadowsocksR，软件收费，订阅普通版一年 130 元人民币左右。网速还可以，YouTube 的 HD 视频基本没压力。

我把我的 ssr 服务器订阅地址分享出来，因为每个账户只能同时在线 3 个人，每月限流 100G，所以我只能让你先关注微信公众号：DeppWXQ。回复 ShadowsocksR，给你我的订阅地址。(对于这种方式，我也很不爽，没办法)。

关于 ShadowsocksR 的客户端下载和使用教程，请看这篇文章：

Wunderlist（奇妙清单）

我们都需要一个 to-do-list，一个待办事项清单。最好能够多终端无缝同步。我觉得 Wunderlist（奇妙清单） 做到了，所以我希望看见这篇文章的你也能用上这款效率软件，可以规划你的工作、学习、私人计划等等。完成一项，划掉一项，很有成就感。

常用使用方式：

  1. 工作四象限清单：重要且紧急（马上去做）；重要但不紧急（有计划去做）；紧急但不重要（授权别人做）；不重要不紧急（尽量不做）。建立四个清单，清晰明了的规划自己每天的工作内容。
  2. 私人相关：如电影、购物、假日计划、学习计划、idea 等清单
  3. 设置提醒时间，清单可重复 (日、周等)

Wox（Windows 效率启动器）

它有丰富的插件可供选择，使用关键字几乎可以完全使用命令操作电脑的大部分功能，当然 Windows 自带的小娜功能也能实现其中一部分。但是没有 Wox 丰富和人性化。下载地址：

选择 Wox - 版本号.exe 下载，下载安装完成后，输入 Setting，将 Language 设为中文。注：此地址包含后文需要的 Everything 和 Python3.0 的安装包，建议一同下载。
常用功能有：搜索文件、网页搜索、翻译、搜索程序、系统命令等等。(输入 setting 查看)

不用记关键字，在搜索栏输入一个字符，Wox 会默认提示。

搜索文件

需要结合 Everything 软件使用。安装完成后直接在 Wox 里输入文件名即可搜索。

网页搜索

可以将常搜索的网站设置关键字。比如：g(Google)、stackoverflow(Stack OverFlow)、translte(谷歌翻译)、amap(高德)、jd(京东) 等。例如：

  1. translte：https://translate.google.cn/#auto|en|{q} (将 com 改为 cn 后不用翻墙)
  2. amap：https://www.amap.com/search?query={q}
  3. jd：https://search.jd.com/Search?keyword={q} (只支持中文)

参考以上设置，可以自定义关键字，如 zh(知乎)、db(豆瓣)

搜索程序

只需要输入程序名，就能为你检索。有的是打开当前正在运行软件，有的是重新打开软件。当部分软件搜索不到时，需要自己手动添加。

添加方式：插件 -> 程序 -> 添加

翻译功能

两种方式翻译：

  1. 使用 translte 关键词打开谷歌网址翻译
  2. 安装有道词典命令，安装教程： http://www.wox.one/plugin/78 (先安装 Python3.0)

系统命令

可以使用 Restart（重启）、Sleep（睡眠）等命令操作电脑。

更多 Wox 命令可以参考官网：http://www.wox.one/

总结

一旦我有知道一些好的软件，我会第一时间安利出来，欢迎持续关注我的微信公众号 (DeppWXQ)。
