Chrome 上价值一百块钱的插件

发表于 2018-08-29 | 更新于 2019-12-19 | 分类于 Software & Website | | 阅读次数

如果你还没用 Chrome 插件，那么我只能说（注意！此处有停顿 ^_^），现在就用吧。这篇文章分享一下我常用的 6 个插件。Chrome 网上应用店 上安装插件需要梯子，需要梯子的小伙伴请看 这篇文章。针对没有梯子的小伙伴，我已将插件离线打包放在了百度网盘上： https://pan.baidu.com/s/1zvrE6GndFTHMKzP-aeQAVg 提取密码：x9i2。

安装插件的方法很简单：在浏览器搜索框中输入 chrome://extensions/ 打开扩展程序页面 (或者右上角三个点 -> 更多工具 -> 扩展程序 )，选中右上角的开发者模式，将下载好的插件拖动到此页面完成安装。

1.OneTab

点击右上角 OneTab 图标可以将你所有已经打开的标签页和合成一个，减少浏览器所占内存。也可以在网页上右键操作。我常用的使用方式是将常用的网址固定在这个页面。类似这样：

2.Google 翻译

点击右上角 Google 翻译图标，点击扩展程序选项中设置主要语言，如 中文 (简体)。常见使用方式：

  1. 将整体的页面翻译成中文（网页右键选择 “翻成中文 (简体)”）。
  2. 网页选中英文点击翻译图标翻译为中文。
  3. 点击右上角插件图标输入英文翻译。
  4. 直接在 https://translate.google.cn/ 上翻译（不用梯子）

3.Tampermonkey

俗称油猴子，是一个用户脚本管理器。我们可以在这个插件中再安装脚本，来实现破解各大主流视频网站的 VIP 视频等操作。
如何安装脚本：

  1. 点击右上角 Tampermonkey 图标，选择获取新脚本；或者直接打开 https://tampermonkey.net/scripts.php
  2. 点击 GreasyFork，搜索下图中的几个插件安装，作用如名称所示，具体使用方法可看脚本描述。

4.Free Download Manager

是不是受够了迅雷的版权受限、限速和各种广告，那你就很需要 Free Download Manager 了。他是一个免费的，支持 BT 磁力下载的软件。它相当于 Internet Download Manage 免费版，后者也很好用，就是只有 30 天试用期，感兴趣的可以自己鼓捣鼓捣。

使用 Free Download Manager 插件前，要在电脑（Window 和 macOS 均可）上下载安装客户端，插件只是用于唤起客户端。

Free Download Manager 客户端下载地址：http://www.freedownloadmanager.org/zh/download.htm

示例：1. 如果你在 胖鸟电影 上下载 我们这一天，点击磁力下载链接，点击打开 Free Download Manager，点击下载。2. 将通过油猴子脚本生成的百度云直链下载 url 复制到 Free Download Manager 中下载。

5.Adblock Plus

用于拦截广告，你可能会想，视频网站的广告能拦截吗？先告诉你答案：可以！非常实用，强烈推荐。拦截视频网站的广告时，会有一会儿等待时间。其他的拦截你可能感知不到。

6.LastPass: Free Password Manager

如果你各个网站密码不一样，经常忘记，那你就很需要这个插件了。使用前需要先注册一个账号，第一次登陆某个网站时，会提醒你是否添加账号密码到 LastPass 中，点击 添加 添加，点击 discard 取消。添加后，当你再次登录时，会自动将账号密码填入输入框。你也可点击输入框 右侧三个点 -> 更多 ->Edit-> 眼睛图标 查看密码。

很多小伙伴可能会担心安全，我觉得没有绝对的安全。一套密码走天下，也不安全。可使用 LastPass 生成安全密码的功能，结合 LastPass 保存，手机安装 LastPass 的 APP 查看密码 (需梯子)。

如何将自己的插件打包保存

输入 chrome://version/，C:\Users\yourName\AppData\Local\Google\Chrome\User Data\Default\Extensions 即是插件的默认安装路径。

扩展程序页面的 OneTab 插件ID对应默认安装路径下的目录名；

扩展程序页面中点击打包扩展程序，将上图 OneTab 插件目录地址复制到扩展程序根目录上

选择打包扩展程序，最后将文件名修改一下。

结语

以上是我常用的 6 个 Chrome 浏览器插件，其他浏览器没有测试过，谷歌大法好，建议使用 Chrome 浏览器。Chrome 上还有更多更好的插件需要我们自己去挖掘体验。还是那句话，多多动手！

ps：没有梯子的小伙伴也可前往这个插件网 ( http://www.cnplugins.com/ ) 下载插件
