我的装修顺序

发表于 2023-12-25 | 更新于 2024-02-27 | | 阅读次数

我开始装修时，对装修的先后顺序不清楚，特别是完全不知道什么时候应该购买什么东西。经历这次装修，发现只要知道这个东西什么时候用，用前买好就行。比如厨房烟道止逆阀，贴砖前买好就行。很多家电可以先不买，但需要确定购买型号，如洗碗机、冰箱、油烟机、马桶、扫地机器人等。这样水电和全屋定制就可以更好设计。大型家电可以活动时购买，推迟发货时间。以下我我装修材料购买 / 施工安装的顺序。

设计阶段

  1. 2022.10.1 全屋量尺寸
  2. 2022.10.6 平面设计优化初稿
  3. 2022.10.7 全屋复尺
  4. 2022.10.11 平面设计优化定稿

拆改阶段

  1. 2022.1.1 拆改工具购买
  2. 2023.1.3 阳台窗户预定
  3. 2023.1.3 开工
  4. 2023.1.3-1.10 拆改（旧墙拆除、砌新墙）
  5. 2023.1.9 除渣
  6. 2023.1.31 铲墙（铲除开发商腻子）

水电阶段

  1. 2023.1.4 网线购买
  2. 2023.2.1 水电定位，水电材料购买
  3. 2023.2.5 水电开槽
  4. 2023.2.6 小区装修过业主推荐新水电师傅
  5. 2023.2.8 新水电师傅进场
  6. 2023.2.14 水电、灯二次设计后，石工补开槽
  7. 2023.2.15 水电穿线完成

泥工阶段

  1. 2023.1.1 看瓷砖
  2. 2023.2.7 小区群找到泥工师傅
  3. 2023.2.8 燃气改表线上申请
  4. 2023.2.11 泥工约看
  5. 2023.2.11 风管机预定
  6. 2023.2.14 燃气改表线上缴费
  7. 2023.2.16 泥工进场批荡（抹灰）做防水。中间不定时购买水泥河沙
  8. 2023.2.20 空调、热水器打孔
  9. 2023.2.20 风管机安装
  10. 2023.2.20 燃气改表
  11. 2023.2.26 墙砖瓷砖胶购买
  12. 2023.2.26 瓷砖送货
  13. 2023.2.28 燃气管预埋
  14. 2023.3.1 开始贴砖
  15. 2023.3.2 阳台窗户安装
  16. 2023.3.4 全屋定制交订金，开始设计
  17. 2023.4.1 泥工贴砖完成

木工阶段

  1. 2023.2.7 刚装修业主推荐木工
  2. 2023.2.11 木工约看
  3. 2023.2.22 买木工材料
  4. 2023.2.24 木工进场，1 天半完成

油工阶段

  1. 2023.2.8 油工，小区中碰见加微信
  2. 2023.4.2 油工工地约看、开材料
  3. 2023.4.2 定室内门，量尺寸
  4. 2023.4.9 油工材料购买
  5. 2023.4.15 刮石膏（底层腻子）
  6. 2023.4.25 全屋定制设计完成出报价
  7. 2023.4.27 找另一家全屋定制
  8. 2023.5.7 全屋定制设计完成开始生产
  9. 2023.5.8 刮第一遍面层腻子
  10. 2023.5.10 安室内木门
  11. 2023.5.17 安窗台石
  12. 2023.5.30 刮第二遍面层腻子
  13. 2023.6.4 厨卫美缝清理与美缝
  14. 2023.6.4 全屋定制送货安装
  15. 2023.6.15 在全屋定制处预定厨房石材
  16. 2023.6.19 安装厨房石材
  17. 2023.6.22 吊顶材料购买
  18. 2023.6.22 看浴室柜
  19. 2023.6.23 浴室柜没货，再看浴室柜
  20. 2023.6.25 浴室柜安装
  21. 2023.6.27 厨卫吊顶安装
  22. 2023.6.28 安厨房、卫生间门
  23. 2023.6.30 腻子打磨
  24. 2023.7.6 刷乳胶漆（天气热，底漆面漆一起一天刷完）
  25. 2023.9.4 开关插座安装后，油漆补缝

家具 / 家电阶段

  1. 2023.6.18 热水器 / 冰箱 / 油烟机 / 智能窗帘 / 开关购买
  2. 2023.6.23 看床 / 沙发 / 床垫
  3. 2023.6.26 热水器安装
  4. 2023.6.30 看灯
  5. 2023.7.13 淘宝买主卧床
  6. 2023.7.14 京东买沙发 / 床垫 / 次卧床
  7. 2023.7.15 开荒保洁
  8. 2023.7.15 地面美缝
  9. 2023.7.19 踢脚线安装
  10. 2023.7.21 洗衣机购买
  11. 2023.7.21 开关灯具安装（3 天）
  12. 2023.7.29 主卧床 / 次卧床安装
  13. 2023.8.5 智能窗帘轨道安装
  14. 2023.8.5 买窗帘
  15. 2023.8.5 沙发送货安装
  16. 2023.8.12 智能马桶购买
  17. 2023.8.13 油烟机安装
  18. 2023.8.13 燃气自闭阀安装
