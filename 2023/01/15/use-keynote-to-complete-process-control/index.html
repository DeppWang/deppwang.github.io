使用 Keynote 完成年会场控

发表于 2023-01-15 | 更新于 2024-06-19 | | 阅读次数

公司年会安排我做年会场控（流程控制），控制场内屏幕和音响，使音频、视频、图片按流程顺序播放，不出现问题。

最开始主要在想用什么方案来实现，尝试过这几种方案：

  1. 使用 WPS 的演示模式，演示模式利用扩展屏控制图片，主屏可以做其它相关工作，使用相关工具控制音频和视频。但在使用 WPS 时，出现过卡死的情况，加上频繁人为操作切换应用大大增加出错风险，最终 pass 了这种方案
  2. 经同事提醒，使用 Keynote，完全用 Keynote 控制音频、视频和图片。选择了这个方案

我使用 Keynote 的一些功能

  1. 视频循环播放，用在了暖场视频和一个节目中
  2. 音频循环播放，用在了活动的背景音乐中
  3. 图片画廊，一个 PPT 中插入多个图片，用在了「你画我猜」的词语切换上
  4. 图片插入音频，使用点按控制播放
  5. 音频音量调整
  6. 音频播放长度设置
  7. 演讲者注释写流程动作提示
  8. PPT 导出为图片
  9. 快捷键控制 PPT

我使用的 Keynote 快捷键，演示模式的下 ？ 键唤出

  1. K 暂停 / 播放影片
  2. J 倒回影片（如果已暂停，按帧倒回）
  3. L 快进影片（如果已暂停，按帧快进)
  4. I 跳到影片开头
  5. O 跳到影片结尾
  6. Shift + ↓前往下一张幻灯片，结束「你画我猜」

个人认为 Keynote 的一些不足

  1. 在演示模式下播放视频没有相关进度提示，我在另一个电脑上看老版的 Keynote 是有的，不知是不是我没有设置来
  2. 使用演示模式时，无法使用其它应用。这个功能不知是不是故意这么设计，避免切换应用导致出现一些未知的问题。应对方式时准备另一个电脑

做 PPT 的一些经验

  • 如果某个环节既需要音频又需要视频，让相关负责人员合成一个，尽量减少人为操作

一些提前准备工作

  1. 电脑屏幕设置不熄灭
  2. 关闭触发角关闭显示器功能
  3. 关闭屏幕保护
  4. 准备 2 个充电器。这个确实用到了，其中一个充电器的转换头落了，还好带了两个
  5. 准备 2 个转接头
  6. 准备一个备用电脑，并如上配置

现场的调试

  1. 重启电脑
  2. 设置扩展器背景为活动背景
  3. 所有要使用软件打开一次，确保从主屏幕打开
  4. 关闭活动无关应用，如微信、飞书
  5. PPT 中音频音量大小调试
  6. 两个电脑都调试好

百密有一疏，遇到的一些问题

  1. PPT 文字丢失。使用一个 PPT 中内容时，为保证质量没有截图而是复制 PPT 图片，没有注意到这页 PPT 是多个图片组合而成，导致 PPT 文字丢失
  2. 背景音乐没有设置重复播放
  3. 临时调高音频设备声音，导致话筒声音失真
  4. 屏幕保护设置没有关闭，年会结束后最后出现了屏保
  5. 背景音乐比较单一，感觉可以更好

一些感想

  1. 年会效果还是不错，作为第一次做场控，对自己的表现还是比较满意，Keynote 作为苹果应用，稳定性确实没得说，没掉链子
  2. 开始时还有一点紧张，但随着活动的进行，就慢慢放松了
  3. 坐在前面，看见大家玩得开心、杯觥交错，心里感觉平和、开心，这就是年会的意义吧
  4. 通过这次经历，接触到了 Keynote，Keynote 很强大，后面 PPT 应该就会用 Keynote 来做了
