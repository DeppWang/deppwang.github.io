分享几个用了就离不开的实用网站

发表于 2019-01-11 | 更新于 2019-12-19 | 分类于 Software & Website | | 阅读次数

1、鸠摩搜书：一个能搜索全网电子书的网站

网址：https://www.jiumodiary.com/

杜撰 slogan：找电子书，来我这里就对了

喜欢看书的小伙伴，是不是觉得买纸质书太贵且不方便，而电子书又在网上到处找不到资源呢！

那你就很需要鸠摩搜书了。他是一个搜书引擎，可以抓取到很多资源，他的宣言可以理解为：“我不生产书，我只是书的抓取工”！

资源下载后，如果你是使用 Kindle 阅读器，或者 Kindle 阅读 App，可以使用邮箱推送，不会请百度。针对没有不用 Kindle 阅读器的小伙伴，建议下载 epub 格式，使用多看阅读 App阅读，多看阅读 App 相比 Kindle 阅读 App，它的黑色纹理背景不伤眼，格式也要好看不少。

从前有个集思会，它黄了 (╥﹏╥)。

2、疯狂影视搜索：一个能搜索全网影视资源的网站

网址：http://ifkdy.com/

杜撰 slogan：超清版我搜不到，枪版我肯定搜得到

现在在单一的视频资源网站上，资源已经很有限了。所以我们需要一个能抓取全网资源的搜索引擎，这样我们不用挨个在每个网站搜索了。

胖鸟电影和人人影视这两个网站也可以收藏，页面美观，资源丰富，谁用谁知道。

有经济能力，还是买各大视频网站会员吧，最新播放影视不用等资源，最新下映院线大片抢先看。

3、WXMarkdown：一款简单易用的微信公众号格式化工具

网址：http://md.barretlee.com/

杜撰 slogan：有了我，公众号和 MarkDown 更配哦

我这个文章就是用的 WXMarkdown 格式化的，喜欢就赶快去用用吧。

4、Server 酱：一款推送通知到微信的工具

网址：http://sc.ftqq.com/3.version

杜撰 solgan：推送通知，简单快速我最棒

在使用分流抢票 (最新版) 时，发现的这个工具，他可以自定义通知内容，当抢到票后，将通知推送到微信上，避免错过订单支付时间。这么好的工具，我怎么现在才发现呢！

我们可以发送一些信息到手机微信上，当电脑上没有使用相关软件时。对于程序员来说，可以在下面这些地方使用：

1、将某些重要的服务器异常，除了在日志中打印，也推送到微信上
2、将某些定时任务执行结果，推送到微信上
3、其他的请查看 Server 酱

5、wikiHow：一个能让所有人学做任何事的平台

网址：https://zh.wikihow.com/% E9% A6%96% E9% A1% B5

杜撰 slogan： 知乎啊，你不行了

原来专业的知识问答，我们都去知乎找答案。但现在知乎很多答案都不是很专业，或者甚至找不到答案，知乎似乎是越来越不靠谱了。

我无意间发现 wikiHow，知识专业，还有人性化的图片，实在是一个首选的问答 (学习) 网站。

6、草莓数码：一个用内部价买苹果正品的网站

网址：http://www.berrydigi.com/index.htm

杜撰 slogan：因为不交税，所以内部价

每个程序员都想拥有一个自己的 Mac，高昂的售价，干瘪的钱包，让我们望而却步。发现周边同事都在这网站买到了内部价的正品 Mac(内部价传言是因为不交税)，所以推荐出来。草莓数码在北京海淀五道口有实体店，建议去店里买。

总结

1、2、3、4 应该都是程序员的个人代表杰作，请鼓掌！ps：如果网址变化，请搜索中文名
